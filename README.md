# Gank.io

一个精简版的gank.io非官方客户端，只有干货。

## 简介

一个精简版的gank.io非官方客户端，只有干货。纯练手项目，熟悉了一下MVP模式和RxJava2和Retrofit2的使用，使用CustomTabs打开网页。
